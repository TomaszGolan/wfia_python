# Imię i nazwisko:  \vspace{20pt}

Niech `dane.txt` zawiera:

```
0123456789
```

1. Jaka będzie zawartość pliku po wykonaniu poniższego skryptu:

```py
dane = open("dane.txt", "r+")

dane.read()

dane.write('abc')

dane.close()
```
